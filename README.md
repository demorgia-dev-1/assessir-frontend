# Assessir Frontend

Turborepo workspace containing three Next.js apps:

- `admin`
- `candidate`
- `assessor`

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start all apps:

```bash
npm run dev
```

3. Or start a single app:

```bash
npm run dev:admin
npm run dev:candidate
npm run dev:assessor
```

## Ports

- `admin`: `3001`
- `candidate`: `3002`
- `assessor`: `3003`
